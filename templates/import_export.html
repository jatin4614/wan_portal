1. Offline Map (MBTiles)
checking…

Upload an offline map file. Two formats are supported:

  • .pmtiles — recommended. A modern single-file format with much better cartography. Vector PMTiles render crisply at any zoom and produce a Google-Maps-quality look.
  • .mbtiles — classic raster tile container. Larger files, fixed-resolution at each zoom level.

Where to get a file: download a region from maps.protomaps.com/builds (PMTiles, free) or build one with the Protomaps app. Suggested bbox for North India: lat 26°–37°N, lng 72°–82°E. 3. Upload Updated Excel (Sync)

Upload a filled-in Excel file to synchronize the system data. Existing data is preserved. The system will:

  • Add rows present in the sheet that aren't already in the database
  • Update existing rows that match by their unique key
  • Remove database rows whose key is not present in the uploaded sheet (redundant data cleanup)
  • A sheet that's missing from your upload leaves that table untouched

Useful as realistic test data. This will erase existing nodes, OFC, equipment, alt-media and radio data.
